Política de Privacidade — Oratio

Última atualização: 6 de março de 2026

1. Informações que Coletamos

O Oratio não coleta dados pessoais de usuários finais. O App opera de forma automatizada e utiliza apenas as credenciais de API fornecidas pelo próprio operador para publicação de conteúdo.

2. Uso das Informações

As credenciais de acesso às plataformas (YouTube, TikTok) são utilizadas exclusivamente para publicação automatizada de conteúdo religioso e nunca são compartilhadas com terceiros.

3. Plataformas de Terceiros

O App integra com plataformas externas (YouTube, TikTok, Google Cloud). Cada plataforma possui sua própria política de privacidade, à qual o operador está sujeito ao utilizar suas APIs.

4. Segurança

Adotamos medidas razoáveis para proteger as credenciais e informações armazenadas localmente. As credenciais OAuth são armazenadas de forma segura no dispositivo do operador.

5. Retenção de Dados

Não retemos dados de usuários. Os vídeos gerados são armazenados localmente e nas plataformas de publicação escolhidas pelo operador.

6. Seus Direitos

Como o App não coleta dados pessoais de terceiros, não há dados a serem solicitados, corrigidos ou excluídos por usuários finais.
