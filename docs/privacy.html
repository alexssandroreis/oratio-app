Política de Privacidade — oratio

Última atualização: 2026-03-11

1. Introdução

Esta Política de Privacidade aplica-se ao aplicativo "oratio" ("Oratio" ou "Oratio App"), disponível em https://github.com/alexssandroreis/oratio-app e outros serviços relacionados fornecidos pelo autor. Esta política descreve como coletamos, usamos, armazenamos e compartilhamos informações pessoais quando você usa o Oratio.

2. Dados que coletamos

  • Dados fornecidos pelo usuário: intenções de oração, preferências de voz, preferências de agendamento (quando aplicável).
  • Dados técnicos/diagnósticos: identificadores de dispositivo, logs de erro, informações de versão do app.
  • Dados de autenticação/integração: quando o usuário conecta contas para upload: tokens OAuth usados apenas para realizar uploads (armazenados localmente no workspace do proprietário e/ou em credenciais configuradas pelo administrador).

3. Finalidade do tratamento

Usamos os dados para:

  • Gerar conteúdo de áudio e vídeo conforme solicitado;
  • Fazer upload de vídeos para contas autorizadas (YouTube/TikTok) quando o usuário autoriza;
  • Melhorar a qualidade do serviço, diagnosticar problemas e gerar relatórios de uso;
  • Comunicar atualizações relevantes e solicitações técnicas ao proprietário da conta.

4. Compartilhamento de dados

Não vendemos dados a terceiros. Dados de integração com plataformas (ex.: YouTube OAuth tokens) são usados exclusivamente para a ação autorizada (upload) e podem ser armazenados no workspace do proprietário. O administrador pode revogar o acesso a qualquer momento.

5. Base legal

O tratamento é realizado com base no consentimento do usuário e na necessidade de execução de serviços contratados (quando aplicável).

6. Retenção

Mantemos dados pelo tempo necessário para as finalidades descritas ou até que o usuário solicite exclusão.

7. Direitos do titular

Você pode solicitar acesso, correção, exclusão ou portabilidade dos seus dados. Para solicitações, contate: privacy@oratio.app

8. Segurança

Implementamos medidas técnicas e organizacionais razoáveis para proteger dados contra acesso não autorizado.

9. Contato

Para dúvidas sobre privacidade, entre em contato: privacy@oratio.app

10. Alterações nesta política

Podemos atualizar esta política periodicamente. A versão mais recente estará disponível no repositório do projeto e a data de revisão será atualizada neste documento.
